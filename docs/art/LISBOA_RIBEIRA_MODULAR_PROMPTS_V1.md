# Lisboa Ribeira Modular Prompts v1

Use these prompts to generate the next assets that will most improve `Lisboa Ribeira`.

## Global Settings

### Base Prompt Prefix
```text
late 15th century Portuguese maritime city, Lisbon Ribeira, circa 1490-1505, HD-2D pixel art look, 3/4 oblique camera, semi-realistic compact proportions, warm Atlantic sunlight, historical realism with controlled stylization, readable tactical RPG environment, coherent world and combat visual language, game-ready, no text
```

### Negative Prompt
```text
modern objects, steampunk, neon, symmetrical fantasy town, generic medieval village, blurry details, washed colors, empty plaza, oversized props, text, logo, watermark, isometric board-game look, front view, top-down view
```

### Output Rules
```text
PNG, transparent background when asset sheet or isolated object, clean separation between elements, nearest-neighbor friendly rendering, high readability at small scale, keep soft warm shadows
```
